Injection Speed Test into Page Served by a Service Worker

How to Test

  1. Open DevTools.
  2. Go to the Application tab in DevTools.
  3. Ensure there is no checkbox next to the "Bypass for network" option.
  4. Go back to the Console tab.
  5. Reload the page.
  6. Make sure that script and scriptlet rules were injected before the inline script at the end.
